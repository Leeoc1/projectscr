주요 관계 요약
 - 사용자(user): 예약과 결제를 여러 개 가질 수 있음.
 - 예약(reservation): 사용자, 상영정보, 예약좌석, 결제와 연결.
 - 상영정보(schedule): 영화, 상영관과 연결되며, 예약을 여러 개 가질 수 있음.
 - 상영관(screen): 지점에 속하며, 좌석과 상영정보를 여러 개 가질 수 있음.
 - 좌석(seat): 상영관에 속하며, 예약좌석을 여러 개 가질 수 있음.
 - 영화(movie): 상영정보를 여러 개 가질 수 있음.
 - 지점(cinema): 지역에 속하며, 상영관을 여러 개 가질 수 있음.
 - 공지사항(notice) 및 FAQ(faq): 독립적인 정보 테이블로 다른 엔터티와 직접 관계 없음.
